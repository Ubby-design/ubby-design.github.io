# Lab 3: Network Firewalls Configuration - Configs

## Overview
This directory contains configuration files and scripts used for **Lab 3 Network Firewalls Configuration**. 
The lab focuses on implementing, testing, and verifying firewall rules in a network environment, simulating real-world enterprise security practices.

These configs are designed to help you:  
- Understand and implement firewall rules.  
- Secure network traffic between hosts and subnets.  
- Practice basic to advanced packet filtering, NAT, and port forwarding.  
- Learn how firewalls interact with other network devices and policies.

# Lab 2: Vagrant VM Provisioning Scripts

## Overview

This directory contains provisioning scripts used to configure the **Lab 2 Vagrant VM**. 
The scripts automate the setup of the VM environment, including; 
package installation, configuration, and initial system hardening, ensuring consistent and repeatable lab setups.

These scripts are designed to be executed automatically by **Vagrant** during VM provisioning or manually for testing and debugging purposes.
